Screen Modes

                            

                                The scoreboard switches modes automatically based on what the CTS console is sending.
                            

                            

                                
Splash  ←  No activity for Intro timeout or Results timeout

                                
↓ New event / heat received

                                
Intro    Shows swimmer names — if no names, goes straight to Running

                                
↓ Any lane starts running

                                
Running  Shows live clock in all active lanes

                                
↓ All lanes finish

                                
Results  Shows final times and podium highlights (name column empty if no names loaded)
